### Current work

At QS (HolonIQ), I architect and maintain a probabilistic hierarchical forecasting platform for international student flows across 250+ destination countries. The system is centred on a **Deep Negative Binomial Mixture Network (DNBMN)** in PyTorch — a modified DPMN (Olivares et al., 2024) with NB emission, entropy regularisation, and a coherence loss on destination totals — with downstream disaggregation to 100+ universities per destination across 5 academic subject areas via a separate 5-layer neural network and IPF matrix-balancing.

Outputs fed the [**QS Global Student Flows Report 2025**](https://insights.qs.com/hubfs/Reports/Global%20Student%20Flows%20Report%202025.pdf), which drew coverage in [*The Economist*](https://www.economist.com/asia/2025/09/25/asian-countries-are-nabbing-a-lot-more-foreign-students), [The PIE News](https://thepienews.com/8-5-million-international-students-by-2030-qs-data-predicts/), and [ICEF Monitor](https://monitor.icef.com/2025/07/new-analysis-forecasts-international-student-mobility-through-2030/).
